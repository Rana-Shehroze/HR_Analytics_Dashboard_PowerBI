# 👨‍💼 HR Analytics Dashboard (Power BI Project)

This project visualizes attrition trends and employee demographics using Power BI, providing actionable HR insights.

## 📊 Key Metrics
- Attrition Rate
- Average Age, Salary, and Years at Company
- Education Background of Employees
- Job Satisfaction by Role
- Top Roles with Highest Attrition
- Gender and Department Breakdown

## 🛠 Tools Used
- Power BI Desktop
- DAX Measures
- Data Modeling
- Data Cleaning using Power Query

## 📂 Files
- `HR_Analytics_Dashboard.pdf` → Final dashboard report export
- HR DataSet

## 📈 Key Visuals
- Attrition KPIs
- Salary Groups Analysis
- Age Group Analysis
- Top 4 High Attrition Roles
- Department-Wise Employee Distribution

## 🧠 Learnings
- Data modeling in Power BI
- Creating dynamic dashboards
- Understanding HR business questions
- Applying filters and drill-downs for better storytelling
